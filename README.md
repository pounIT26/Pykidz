# Pykidz-Vibe-coding-web-for-teach-and-practice-Python-for-kids
  Hi there, this is pykids this website was create to help kids learn python using interactive challenge and gain point ( we still working on it even tho vibe code give good result but it was not looking good and have bug and unstable to push out as feature) 

#My goal: 
  for this project is help kids learn and have fun to satisfy vast demographic in the classroom challenge, I was to cut down on cost as much as possible since i dont have much money.

#Challenge: 
  Kids in classroom have various level some goes faster than other some already know concept and need harder challenge

#Solution: 
  Pykidz that create challenge list and whenever you finish a problem you gain point or have interactive feature that make your browser or coding website more vibrant or more fun like adding pet or adding picture as randomize time as reward to make you want to do it even more.
